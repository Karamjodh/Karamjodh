## 📂 Featured Builds

<details open>
<summary><b>🧠 FER_XAI — Explainable Facial Expression Recognition</b></summary>
<br/>

7-class facial expression recognition with post-hoc explainability via **LIME** and **SHAP**. Two CNN backbones — **ResNet-50** and **EfficientNet-B0** — trained and benchmarked on FER2013 and RAF-DB, with pixel- and region-level explanations generated for every single prediction.

- 🏆 Best result: **84.36% accuracy** (ResNet-50 on RAF-DB)
- 🔍 Every prediction comes with a visual "why" — not just a label
- 🛠️ Stack: `PyTorch` · `LIME` · `SHAP` · `AMP training`

**[→ View Repository](https://github.com/Karamjodh/FER_XAI)**

</details>

<summary><b>🎣 Phishing-Detection — Real-Time URL Threat Classifier</b></summary>
<br/>

An ML-powered system that extracts **30+ security-relevant features** from any URL — structural, content-based, and domain-based — to classify it as legitimate or malicious in real time. Fully containerized and deployed on AWS via an automated CI/CD pipeline.

- ⚡ ~95%+ accuracy on held-out test data
- ☁️ Dockerized, deployed on AWS EC2, tracked with MLflow + DagsHub
- 🛠️ Stack: `Flask` · `Scikit-learn` · `Docker` · `AWS` · `MongoDB`

**[→ View Repository](https://github.com/Karamjodh/Phishing-Detection)**
